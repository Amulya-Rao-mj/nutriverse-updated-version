A modern, personalized nutrition website that helps users track their profile and get meal suggestions based on their diet preferences.

## Features

- **User Authentication**: Secure login and signup system
- **Editable Profile**: Manage your personal information including:
  - Name
  - Age
  - Gender
  - Diet Type (Vegetarian, Non-Vegetarian, or Vegan)
  - Height and Weight
  - Automatic BMI calculation
- **Personalized Meal Suggestions**: Get meal recommendations based on your selected diet type
- **Modern UI**: Beautiful, responsive design that works on all devices

## How to Use

1. **Sign Up**: Create a new account with your email and password
2. **Login**: Use your credentials to access your dashboard
3. **Complete Profile**: Fill in your profile information including diet type
4. **View Meals**: Navigate to "Meal Suggestions" to see personalized meal recommendations

## Technology Stack

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript
- LocalStorage for data persistence

## Getting Started

Simply open `index.html` in your web browser. No build process or server required!

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.

## Notes

- Data is stored locally in your browser using localStorage
- Clear browser data to reset all stored information
- This is a frontend-only application - no backend server required
